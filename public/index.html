Timestamp Microservice
Enter either a date or unix timestamp and you will get back both the date and unix time
For example,
https://timestamp-joshlevy89.c9users.io/December16,2015 will return {"unix":1450224000,"natural":"December 16, 2015"}
https://timestamp-joshlevy89.c9users.io/1450224000 will return {"unix":1450224000,"natural":"December 16, 2015"}
